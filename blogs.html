The scope of var, let, and const

Var:It i global scope , when you declare in var you can access this variable from any where outside of function

Let and Const: when you declare Let and Const inside any function {}. They are not accessible outside of that block.

Use cases of null and undefined

Undefined: It is a default Value Variables that are declared but not assigned a value are automatically set to 'undefined' . when you call a function with parameters , if parameter is missing it will show the missing parameter as 'undefined'. also in object properties you want to access an object that does not exist it will show 'undefined'

Null: we can explicitly set a variable or object property to 'null' to indicate that it intentionally has no value. It's often used to clear the value of an object property or variable. we can use 'null' to check if a value is intentionally missing, while 'undefined' typically represents unintentional absence.

What do you mean by REST API?

A REST API is like a set of rules that computers use to talk to each other on the internet. It follows a specific way of doing things called REST, which stands for "representational state transfer." Think of it like a language that computers use to have conversations. When a computer follows these REST rules, we call its API "RESTful."
